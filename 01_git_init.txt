
On initialise le repository ou proget Git dans le dossier demogit. Pour cela, on va utiliser la commande git init. Cette commande permet d'initialiser un repository Git dans le dossier courant.
cela va créer un dossier caché .git dans le dossier demogit. Ce dossier contient toutes les informations sur le repository Git. Il ne faut surtout pas le supprimer.
Il est possible de le cacher en utilisant la commande : git config core.hideDotFiles false.
Pour afficher les fichiers cachés, on utilise la commande : ls -a / Sur windows : dir /a
